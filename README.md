# Titanic Dataset EDA

## Overview

Performed Exploratory Data Analysis (EDA) on the Titanic Dataset using **Pandas** and **NumPy** to understand factors affecting passenger survival.

## Dataset

* Rows: 891
* Columns: 12
* Target Variable: `Survived`

## Tools Used

* Python
* Pandas
* NumPy

## EDA Process

### 1. Data Understanding

* Inspected dataset structure
* Checked data types and statistics

### 2. Data Quality Check

* Identified missing values
* Checked duplicates
* Examined unique values

### 3. Statistical Analysis

Used NumPy to calculate:

* Mean
* Median
* Standard Deviation
* Variance
* Percentiles

### 4. Feature Analysis

Analyzed:

* Age
* Fare
* Sex
* Pclass
* Embarked

### 5. Relationship Analysis

Studied how different features affected survival using:

* GroupBy
* Crosstab
* Correlation Analysis

## Key Findings

* Females had a much higher survival rate than males.
* First-class passengers survived more often than third-class passengers.
* Higher fare was associated with higher survival probability.
* Age had a weaker impact compared to gender and passenger class.
* Socioeconomic status significantly influenced survival outcomes.

The analysis showed that **Sex, Pclass, and Fare** were the most influential factors affecting survival. This project provides a strong foundation for Data Analysis, Data Science, and Machine Learning projects.
